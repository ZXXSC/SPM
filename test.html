What is SmartDoc?

    Smartdoc is a command line program that generates a cross-linked software document in HTML format, with the following command line UI prototype.
    And it has two inputs that Software Requirements Specification in a plain text file (e.g., srs.txt) and a source code file (e.g., code.py).
   The output will be srs.html and code.html, which are meant to help software maintainer to easily find a corresponding requirement (in srs.html) by clicking a link in the converted source code, code.html.

How to use SmartDoc?

    First, type the command line in cmd, which like this.
python SmartDoc.py srs.txt code.py

Then press the 'Enter' key to run it.
    Second, you will see the two html files are in the same folder which the original files are.
    Third, you can click the hyperlink to find the corresponding function and code.
